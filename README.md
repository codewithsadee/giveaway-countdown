# Giveaway Countdown

 A countdown clock which can be used when a new product is coming out or a sale is about to end on an ecommerce site.

 #### Key Concepts Covered
 * document.querySelector() 
 * addEventListener()
 * array 
 * setTimeout()
 * getDate()
 * getHours()
 * getMinutes()
 * getSeconds()
